z-index

che kala mung pa z-axis bnde yo image move kol gwaro. z-index pa position static value bnde kar na kai mung ba pehla positioning kao absolute, relative ya fixed.
Os mung gwaro che 01 tolo pass rawalo no z-index ba use kao.
By default che d z-index value auto v kho auto sare sa affect na v.
